Java Script Fundamentals

  • when we create a function by default 'this' will pointing to global environment from which the function is created
    var temp = function () { alert(this); }; temp();
  • When we create an object with new keyword 'this' reference will be set to the object that is created by new
    javascript creates a brand new object for us
    javascript sets the 'this' method inside initialization function to the newly create object
    var temp = new function () { alert(this); };
  • javascript ignores return value of the initialization function and then returns the created object
    var temp = new function () { this.someVariable='testing'; return true; }; alert(typeof temp);
  • Calling a method on the object will set the 'this' ref inside the method with the object from which the method is called.
    var temp = { sampleMethod: function () { alert(this); } } temp.sampleMethod();
  • javascript has function types
    var temp = function() { }; alert(typeof temp);
  • When an object is created with 'new' keyword the _proto_ hidden variable is populated with prototype of the initializing function.
    var Temp = function() { }; Temp.prototype.someMethod = function () {}; var temp = new Temp(); alert(typeof Temp); alert (typeof temp);
